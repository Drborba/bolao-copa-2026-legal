1. INTRODUÇÃO E COMPROMISSO COM A PRIVACIDADE

A presente Política de Privacidade descreve como [Nome da Empresa/Desenvolvedor] ("nós", "nosso" ou "Aplicativo") coleta, usa, armazena e protege os dados pessoais dos usuários ("você", "seu") do aplicativo Bolão Copa 2026.

Nosso compromisso é garantir transparência, segurança e respeito aos seus direitos, em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018);
  • Marco Civil da Internet (Lei nº 12.965/2014);
  • Melhores práticas internacionais de privacidade.

Ao usar o Aplicativo, você consente com a coleta e uso de dados conforme descrito nesta Política.

2. DADOS COLETADOS

2.1 Dados Fornecidos Diretamente pelo Usuário

2.1.1 Cadastro Tradicional (Email e Senha)

  • Email: Para autenticação, recuperação de senha e comunicações essenciais;
  • Senha: Armazenada de forma criptografada (hash bcrypt/argon2) – nunca em texto simples;
  • Nickname/Username: Nome de exibição nos rankings e bolões (obrigatório);
  • Nome Completo: Opcional, para personalização do perfil.

2.1.2 Login Social (Google/Facebook)

Ao optar por autenticação via redes sociais, coletamos apenas os dados autorizados por você:

  • Google: Email, nome, foto de perfil do Google;
  • Facebook: Email, nome, foto de perfil do Facebook.

Nota Importante: Não acessamos suas mensagens, contatos, posts ou outras informações privadas das redes sociais. A coleta é limitada ao escopo de autenticação OAuth 2.0.

2.1.3 Dados de Perfil

  • Foto de Perfil (Avatar): Imagem enviada para personalização, armazenada em nosso servidor (Supabase Storage);
  • Preferências de Notificação: Configurações sobre quais alertas deseja receber.

2.1.4 Dados de Uso do Aplicativo

  • Palpites de Jogos;
  • Bolões Criados/Participados;
  • Estatísticas de Desempenho;
  • Palpites de Ouro;
  • Interações Sociais.

2.2 Dados Coletados Automaticamente

2.2.1 Dados Técnicos e de Dispositivo

  • Identificador de Dispositivo;
  • Modelo e Versão do Dispositivo;
  • Sistema Operacional;
  • Versão do Aplicativo;
  • Endereço IP;
  • Estado de Conexão de Rede.

2.2.2 Tokens de Autenticação e Sessão

  • JWT (JSON Web Token);
  • Refresh Token;
  • Push Notification Token.

2.2.3 Logs de Erro e Desempenho

Coletamos automaticamente logs de erro (crash reports) e logs de depuração.

⚠️ ATENÇÃO – POSSÍVEL VIOLAÇÃO DE PRIVACIDADE:
Os logs de depuração podem conter dados sensíveis como email, IDs de usuário ou mensagens de erro detalhadas. É CRÍTICO implementar sanitização de logs antes do envio para produção.

2.2.4 Dados de Localização (LIMITADO)

O Aplicativo NÃO coleta localização geográfica precisa via GPS.

3. FINALIDADE DO TRATAMENTO DE DADOS

Dado Coletado Finalidade Base Legal (LGPD)
Email Autenticação, recuperação de senha Execução de contrato (Art. 7º, V)
Push Notification Token Envio de alertas Consentimento (Art. 7º, I)
Logs de Erro Correção de bugs Legítimo interesse (Art. 7º, IX)

4. COMPARTILHAMENTO DE DADOS COM TERCEIROS

Compartilhamos dados exclusivamente para operação do Aplicativo com:

  • Supabase: Infraestrutura de backend;
  • Expo: Notificações push;
  • Google (OAuth/AdMob): Autenticação e publicidade futura;
  • Facebook (OAuth): Login social.

Não vendemos, alugamos ou comercializamos seus dados pessoais.

5. ARMAZENAMENTO E SEGURANÇA DOS DADOS

  • Criptografia em trânsito (HTTPS/TLS 1.3);
  • Criptografia em repouso (AES-256);
  • Hash bcrypt para senhas;
  • Row-Level Security (RLS);
  • Backups automáticos diários.
⚠️ Nenhum sistema é 100% seguro. Utilize senha forte e mantenha seu dispositivo atualizado.

6. COOKIES E TECNOLOGIAS SIMILARES

Utilizamos armazenamento local (AsyncStorage), tokens JWT e push tokens para funcionamento do Aplicativo.
