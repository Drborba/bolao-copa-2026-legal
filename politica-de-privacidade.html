2. DADOS COLETADOS

2.1 Dados Fornecidos Diretamente

Autenticação: E-mail, nickname e senha (armazenada via hash criptográfico).

Login Social: E-mail, nome e foto de perfil autorizados via Google ou Facebook através de protocolo OAuth 2.0.

Perfil e Uso: Fotos de avatar enviadas para o Supabase Storage, palpites de jogos, estatísticas de desempenho e participações em bolões.

2.2 Dados Coletados Automaticamente

Técnicos: Identificador do dispositivo, modelo, sistema operacional, versão do app e endereço IP.

Sessão: Tokens JWT (JSON Web Token), Refresh Tokens e Push Notification Tokens.

Logs: Relatórios de erro e desempenho para correção de bugs.

4. COMPARTILHAMENTO COM TERCEIROS

Compartilhamos dados estritamente necessários para a operação com:

  • Supabase: Infraestrutura de banco de dados e autenticação;
  • RevenueCat: Gestão de assinaturas e compras In-App;
  • Google (AdMob/OAuth): Autenticação e serviços de publicidade;
  • Facebook: Autenticação via login social.

5. SEGURANÇA E ARMAZENAMENTO

Proteção: Criptografia em trânsito (HTTPS/TLS 1.3) e em repouso (AES-256).

Integridade: Implementação de Row-Level Security (RLS) no Supabase para garantir que usuários acessem apenas seus próprios dados.

Localização: Os dados podem ser armazenados em servidores nos EUA ou Europa via Supabase/AWS.

6. DIREITOS DO TITULAR (LGPD)

Você possui o direito de solicitar o acesso, retificação, exclusão ou portabilidade de seus dados a qualquer momento. A exclusão da conta pode ser realizada diretamente nas configurações do Aplicativo.
